This application is a cross platform development to change filenames on a given folder using a CVS (like excel) file.

Just click on "Choose folder" under Directory label and select a folder with files to rename.

Click on "Generate CVS File!" to download a CVS with columns "Path, Extension, OldName and NewName".

Column NewName must be filled with new filenames.

To rename click on "Choose file" under CVS file, select filled CVS and click on "Change names!" to change all filenames on folder.
